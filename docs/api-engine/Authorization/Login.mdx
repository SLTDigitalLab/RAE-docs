# Login

Login to your account to generate JWT.

<Tooltip color="#3abe25">Method: POST</Tooltip>
<div style={{ marginBottom: "20px" }}></div>

```bash
https://engine.raccoon-ai.io/auth/login
```

## Request Body

| Key      | Data Type | Required | Description                                        |
| -------- | --------- | -------- | -------------------------------------------------- |
| email    | string    | true     | Email of the user.                                 |
| password | string    | true     | Password for the account. (More than 8 characters) |

### Types

```json
{
  "email": <string>,
  "password": <string>
}
```

### Sample

```json
{
  "email": "user@raccoon-ai.io",
  "password": "XXXXXXXX"
}
```

## Reponse Body

| Key     | Data Type | Description                                                                         |
| ------- | --------- | ----------------------------------------------------------------------------------- |
| success | boolean   | Indicate the success of the request                                                 |
| msg     | string    | Message indicators                                                                  |
| result  | json      | Result, only set if success is true. Contains information of the newly created user |
| gen_at  | timestamp | Generated timestamp                                                                 |

### Types

```json
{
    "success": <bool>,
    "msg": <string>,
    "result": <json>,
    "gen_at": <timestamp>
}
```
