# Getting Started

## Welcome to the Raccoon AI Engine documentation!

This guide will walk you through the process of getting started with Raccoon AI Engine,
a powerful Machine Learning as a Service (MLaaS) platform that allows you to train
and deploy machine learning models easily.

### Step 1: Sign up and Obtain API Key

To begin using Raccoon AI Engine, you'll need to sign up for an account. Visit the Raccoon AI Console website at https://console.raccoon-ai.io and create a new account by following the registration process. Once registered, you can generate an API key, which is required for making API requests to the Raccoon AI Engine.
